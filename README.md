# 剛好學

## 專案描述
「剛好學」是一個基於網頁的教育應用程式，旨在提供學生互動、課堂管理和同儕互評等功能。專案結合了前端網頁技術與後端資料庫服務，支援教師管理課程與學生參與。

## 主要功能
- **學生回應系統**: 記錄與管理學生的學習回應。
- **同儕互評**: 支援學生之間的作業或專案互評機制。
- **課堂管理**: 建立、管理和組織不同的課程與班級。
- **教師密碼保護**: 針對教師功能提供安全存取。
- **Firebase 整合**: 可能用於即時資料同步或使用者認證。

## 使用技術
- **前端**: HTML, JavaScript (推測)
- **後端**: PocketBase (作為輕量級後端資料庫和認證服務)
- **雲服務**: Firebase (推測用於特定功能，如即時資料或認證)

## 環境設定與執行

### 1. PocketBase 設定
本專案使用 PocketBase 作為後端。
- 確保 `pocketbase.exe` 檔案存在於專案根目錄。
- 執行 `start.bat` 檔案來啟動 PocketBase 服務。這將會啟動一個本地伺服器，並管理 `pb_data` 中的資料。

### 2. 網頁應用程式
PocketBase 啟動後，您可以透過瀏覽器訪問以下檔案來使用應用程式：
- `剛好學V2.4-firebaseX-教師密碼tpet版.html`: 應用程式的主要入口點，可能包含教師相關功能。
- `pb_public/index.html`: 可能為公開的首頁或學生入口。

## 如何開始
1.  **啟動 PocketBase**: 雙擊執行 `start.bat`。
2.  **開啟應用程式**: 在您的網頁瀏覽器中打開 `剛好學V2.4-firebaseX-教師密碼tpet版.html` 或 `pb_public/index.html`。
